Detailed Description

Rapport détaillé des opérations de pré-traitement effectuées.

Permet de tracer toutes les transformations appliquées aux données et de fournir un retour détaillé à l'utilisateur via l'API.

Note
Ce rapport est rempli progressivement pendant le traitement par le DataPreprocessor. Il doit être initialisé avec le nombre de lignes en entrée avant le traitement.
Le rapport est conçu pour être facilement sérialisable en JSON via les DTOs Oat++ pour l'exposition via l'API REST.
Utilisation typique :
PreprocessingReport report;
report.setInputRowCount(dataset.getRowCount());
// ... traitement ...
report.incrementOutliersRemoved(removed);
report.setOutputRowCount(dataset.getRowCount());
