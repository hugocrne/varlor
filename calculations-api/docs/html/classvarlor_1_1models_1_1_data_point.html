Detailed Description

Représente une ligne individuelle du jeu de données.

Permet un accès rapide aux champs par leur nom via une table de hachage. Chaque champ peut contenir une valeur de type variant (numérique, texte, booléen, null). Un conteneur MetaInfo associé permet de tracer les opérations via la section _meta.

Note
Complexité temporelle :
  • getField(), hasField(), setField() : O(1) en moyenne
  • removeField() : O(1) en moyenne
Les valeurs sont stockées dans un std::variant, permettant un stockage type-safe sans overhead de pointeurs.
Pour vérifier l'existence d'un champ avant accès, utiliser hasField() ou vérifier que getField() retourne une valeur.
