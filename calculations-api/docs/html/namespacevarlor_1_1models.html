Type variant représentant une valeur de champ.

Peut contenir :

  • double : valeur numérique
  • std::string : valeur texte
  • bool : valeur booléenne
  • std::nullptr_t : valeur null (représentée par nullptr)

Enumeration Type Documentation

◆ FieldType

enum class varlor::models::FieldType
strong

Énumération des types de données supportés pour les champs.

Utilisée pour identifier automatiquement le type de chaque colonne lors de la phase de détection de schéma par le DataPreprocessor.

Note
Cette énumération est utilisée pour la détection de schéma et peut être étendue à l'avenir pour supporter d'autres types (dates, catégories, etc.).
